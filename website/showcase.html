Core Protocol Specs

Architected for light footprint, high speed, and direct socket routing without centralized overhead.

🛡️

AES-256-GCM Encryption

Optional payload sealing via `P2P_SECRET` env. All CHAT packet payloads are encrypted on transmission and verified upon receipt using authenticated encryption.

⛓️

1-Hop Message Relay

Route through intermediate peers. When destination node is behind NAT or direct TCP fails, a mutual peer acts as a relay, forwarding the stream and returning `RELAY_ACK` back to source.

📁

File Transfer Protocol

Chunked base64 file sharing. Files are read into binary buffers, segmented into 16KB blocks, wrapped in `FILE_PART` envelopes, and reassembled in sequence at the destination.

💾

SQLite Store & Forward

A built-in transactional outbox handles flaky connections. Messages failing direct TCP delivery are stored in the peer's SQLite file and flushed once the destination peer is online.

📣

Broadcast Flooding

Peer-to-peer network-wide flooding. Utilizes distinct message IDs to prevent duplicates and decrementing TTL counters to avoid network congestion.

👥

Local Fan-out Groups

Create local group sessions and broadcast message objects concurrently to all active socket channels in the group for synchronous group conversation.

Quick Start CLI Guide

Launch the system locally in less than a minute using Windows PowerShell.

Clone the repository and install dependency layers:

git clone https://github.com/Duyduy77/P2P-chat.git
cd P2P-chat/peer-node
npm install

Run the Tracker (Bootstrap server) on port 3020:

cd bootstrap-server
$env:BOOTSTRAP_PORT="3020"
node server.js

In a new terminal, launch Peer "Alice" on port 4101:

cd peer-node
$env:PEER_ID="alice"
$env:LISTEN_PORT="4101"
$env:BOOTSTRAP_URL="http://127.0.0.1:3020"
$env:WEB_PORT="8081"
node peer.js

In another terminal, launch Peer "Bob" on port 4102 and send a chat to Alice:

cd peer-node
$env:PEER_ID="bob"
$env:LISTEN_PORT="4102"
$env:BOOTSTRAP_URL="http://127.0.0.1:3020"
$env:WEB_PORT="8082"
node peer.js

# Inside Bob's console, send a direct chat:
> send alice Hello Alice, I am Bob!
